# ORNAS Brand Assets

This directory contains the official ORNAS brand assets generated by the automated asset pipeline (`npm run generate:icons`).

## Directory Structure

```
branding/
├── source/
│   ├── official_logo.svg      (Filled master, Accent)
│   └── official_outline.svg   (Outline master, Accent)
├── exports/
│   ├── png/                   (Transparent raster exports, sizes 16-1024)
│   │   ├── ornas-mark-accent-*.png
│   │   ├── ornas-mark-dark-*.png
│   │   ├── ornas-mark-white-*.png
│   │   ├── ornas-outline-accent-*.png
│   │   ├── ornas-outline-dark-*.png
│   │   └── ornas-outline-white-*.png
│   ├── svg/                   (Vector exports, all color variants)
│   ├── ico/                   (Windows multi-resolution icon)
│   ├── icns/                  (macOS iconset directory)
│   └── favicon/               (SVG + ICO for web)
├── previews/
│   └── brand-preview.html     (HTML verification sheet)
└── archive/                   (Deprecated legacy branding assets)
```

## How to Regenerate Assets

If the source SVGs are modified, you can regenerate the entire 81-asset suite:

```bash
npm install
npm run generate:icons
```

This will automatically re-render all PNGs, compile the Windows `.ico`, update the Tauri build icons (`src-tauri/icons/`), update the public favicons, and regenerate the HTML preview sheet.
